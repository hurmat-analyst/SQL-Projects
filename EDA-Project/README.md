# Exploratory Data Analysis(EDA) - World Layoffs

**Project Overview:**

This project performs the exploratory data analysis on the cleaned layoffs dataset. The goal is to discover trends, patterns, and 
key insights from the data.

**Tools & Concepts Used:**
- SQL
- Aggregate Functions
- Window Functions
- Tren Analysis by Year, Month, Industry, Company, Stage

**Key Steps:**
1. Analyzed layoffs distribution by company and industry
2. Calculated yearly and monthly trends
3. Ranked top companies by total layoffs
4. Identified pattern for stage-wise layoffs

**Key Insights:**
- Top companies with highest layoffs
- Industry trends and yearly/monthly patterns
- Data ready for visualization or reporting
