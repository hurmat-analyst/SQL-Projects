# Data Cleaning Project - World Layoffs

**Project Overview**
This project demonstrates how raw layoffs data was cleaned and prepared for analysis.The goal is to remove duplicates, standardize data,
handle null and blank values and create a clean dataset ready for further analysis.

**Tools & Concepts Used:*
- SQL
- CTE's
- ROW_NUMBER(), RANK(), DENSE_RANK()
- UPDATE & DELETE statements
- Handling NULL & Blank values
- Removing unnecessary columns
