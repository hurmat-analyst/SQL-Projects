# Data Cleaning Project - World Layoffs

**Project Overview:**

This project demonstrates how raw layoffs data was cleaned and prepared for analysis.The goal is to remove duplicates, standardize data,
handle null and blank values and create a clean dataset ready for further analysis.

**Tools & Concepts Used:*
- SQL
- CTE's
- ROW_NUMBER(), RANK(), DENSE_RANK()
- UPDATE & DELETE statements
- Handling NULL & Blank values
- Removing unnecessary columns

**Steps Taken:**
1. Imported raw dataset into table
2. Removed duplicates
3. Standardize text and numeric data
4. Handled NULL and Blank values in critical columns
5. Deleted unnecessary columns

 **Key Insights:**
 - Cleaned dataset ready for analysis
 - Top companies with layoffs can be identified
 - Data now suitable for exploratory data analysis(EDA)
